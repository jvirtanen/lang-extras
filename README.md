Lang Extras
===========

Lang Extras contains extensions to Java Lang API.


Download
--------

Add a Maven dependency to Lang Extras:

    <dependency>
      <groupId>org.jvirtanen.lang</groupId>
      <artifactId>lang-extras</artifactId>
      <version><!-- latest version --></version>
    </dependency>
